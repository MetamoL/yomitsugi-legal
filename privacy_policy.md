# Yomitsugi プライバシーポリシー

**最終更新日: 2026-07-11**

本ポリシーは、Metamol（以下「当方」）が提供するアプリ **「Yomitsugi」**（以下「本アプリ」）における、利用者の情報の取り扱いについて説明します。

---

## 1. 収集する情報

### (a) 端末内に保存される情報（当方のサーバーへは送信しません）

> ※ ただし、下記 **(e) 新着更新のお知らせ（プッシュ通知）** にご同意いただいた場合は、お気に入り作品の**識別子のみ**例外的に当方のサーバーへ送信されます。既読状態・しおり位置・アプリ設定など、それ以外の端末内データは引き続き送信されません。

- お気に入り作品、ウォッチリスト、既読状態、しおり位置、アプリの各種設定。
- これらは利用者の端末内に保存されます。**iCloud 同期を有効にした場合**、利用者自身の iCloud（Apple のサービス）に保存され、当方はその内容にアクセスしません。

### (b) 利用状況の分析（アナリティクス）
- 機能改善のため、画面の閲覧や機能の利用回数などの**利用状況**を収集します（分析基盤 **PostHog** を利用）。
- アプリの安定化のため、**クラッシュ・エラー情報（診断）**も同じ **PostHog** で**匿名**で収集します（同じく下記のオプトアウトで停止できます）。
- 氏名・メールアドレス等、**個人を直接特定する情報（PII）は収集しません**。端末ごとの匿名 ID で集計し、個人プロフィールの作成やセッション録画は行いません。
- 収集は**公開版アプリでのみ**行い、**アプリの設定からいつでも停止（オプトアウト）できます**（既定は有効）。

### (c) 購入情報（課金機能を利用する場合）
- サブスクリプション等の購入は **Apple（App Store）** を通じて処理され、購入状態の判定に **RevenueCat** を利用します。
- クレジットカード番号等の**決済情報は Apple が管理**し、当方は受け取りません。

### (d) 外部サイトの表示（WebView）
- 本アプリは、小説の閲覧時に「**小説家になろう**」（株式会社ヒナプロジェクト運営）のウェブサイトを表示します。
- 当該サイト閲覧に関する情報の取り扱いは、**同サイトのプライバシーポリシー**に従います。当方は当該サイトを運営していません。

### (e) 新着更新のお知らせ（プッシュ通知）で送信する情報
- 本アプリは、お気に入り作品の更新をプッシュ通知でお知らせする機能を提供しています。この機能は、アプリ内に表示される同意画面で**明示的に同意した場合にのみ**有効になります。同意しなくても、アプリの他の機能はすべてご利用いただけます（通知は端末内のみで完結するベストエフォート方式になります）。
- 同意すると、以下の情報を当方のサーバー（**Amazon Web Services・東京リージョン**で運用）へ送信します。
  - お気に入り作品の識別子（作品ID）のリスト（ミュート中の作品を除く）
  - 通知の配信に使う端末トークン（Firebase Cloud Messaging〔FCM〕の登録トークン）
  - 端末ごとに生成される**匿名の識別子**（アカウント登録は不要で、氏名・メールアドレス等の個人情報とは紐付きません）
  - 通知の頻度・夜間通知の可否などの通知設定
- これらの情報は、**お気に入り作品の更新をプッシュ通知でお知らせする目的にのみ**利用します。分析・広告・第三者への販売等、それ以外の目的には利用しません。
- 通知の配信には、Googleが提供する **Firebase Cloud Messaging（FCM）** を利用します。これに伴い、通知トークンはGoogleを経由して配信されます。Googleにおける取り扱いは Google のプライバシーポリシーに従います。
- 削除:
  - アプリ内でこの通知をオフにすると、上記の情報は当方のサーバーから速やかに削除されます。
  - アプリを起動しないまま**90日間**が経過した端末の情報は、自動的にサーバーから削除されます。
  - 上記に加えて、下記5の問い合わせフォームからも削除をご依頼いただけます。
- 本項の同意・停止は、上記(b)の**PostHog分析のオプトアウト設定とは独立**しています。一方の設定を変更しても、もう一方には影響しません。

## 2. 利用目的
- 本アプリの機能提供、品質・利便性の改善、不具合の調査のため。
- プッシュ通知（1(e)）に関する情報の利用目的は、新着更新の通知配信のみに限定します。

## 3. 第三者提供
- 法令に基づく場合を除き、収集した情報を第三者に販売・提供しません。
- 上記の分析・課金・表示・お問い合わせ・プッシュ通知配信のため、**Apple／PostHog／RevenueCat／株式会社ヒナプロジェクト／Google（問い合わせフォーム、Firebase Cloud Messaging）／Amazon Web Services（プッシュ通知配信サーバーの運用基盤）** 等のサービスを利用します。各社の取り扱いは各社のポリシーに従います。
- 問い合わせフォームにご入力いただいた内容は、お問い合わせ対応の目的にのみ利用します（フォームは Google フォームを通じて処理されます）。
- プッシュ通知（1(e)）でお預かりする情報は、Google（Firebase Cloud Messaging）および Amazon Web Services を通じて処理します。これらの事業者は当方の指示に基づいてのみデータを取り扱う**処理者**であり、独自の目的（広告等）での利用はしません。

## 4. 情報の管理
- 端末内データは利用者の管理下にあります。**アプリを削除すると端末内データは削除**されます（iCloud 上のデータは Apple の仕組みに従います）。
